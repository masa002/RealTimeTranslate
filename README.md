# RealTimeTranslate

Installation
========
Install the module from PyPI
```
pip install -r requirements.txt
```

Usage
========
```
python Translate.py
```

Config
========
```
- API_KEY -
key 		: 	Deepl API Key.

- TRANSLATE_SETTINGS -
source_lang 	: 	The language you speak.
target_lang	:	Target Language.

- TALK_SETTINGS -
speed		:	Talk Speed.
hide		:	Hide Softalk.
```
